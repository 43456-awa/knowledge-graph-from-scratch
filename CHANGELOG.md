# 更新日志

所有重要更改都将记录在此文件。

格式基于[Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [1.0.0] - 2026-07-06

### 新增
- PDF内容提取模块
- NLP处理模块（实体识别、关系抽取）
- 知识图谱构建与存储（NetworkX）
- 查询引擎（实体查询、路径查询、子图查询）
- 推理引擎（传递性、对称性、组合、相似性推理）
- 智能问答API（FastAPI）
- 数据集构建工具
- 完整数据集（9,335实体，26,495关系，34,459三元组）
- 单元测试
- 部署脚本

### 技术栈
- Python 3.9+
- FastAPI
- NetworkX
- jieba
- PyMuPDF
- ECharts

### 数据集
- 从《从零构建知识图谱》书籍提取
- 包含概念、技术、工具、人物、组织等实体类型
- 提供JSON、CSV、TransE训练格式
